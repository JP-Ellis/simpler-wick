Simpler-Wick

This package provides simple way of inserting Wick contractions.

If you have any suggestions or have found any bugs, please feel free create a new issue or pull request here on Github.

Installation

This package is not currently offered on CTAN as it is just a personal project of mine; however, if enough people find it useful, I will look into making it available through CTAN.
